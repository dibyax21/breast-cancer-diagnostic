# AI-Driven Breast Cancer Diagnostic Portal

This repository contains a clinical decision-support portal powered by parallel Machine Learning (Random Forest) and Deep Learning (Multi-Layer Perceptron) models.

## How to Run Locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the web server:
   ```bash
   python app.py
   ```
   Open `http://127.0.0.1:7860` in your web browser.
